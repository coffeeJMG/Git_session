# talent sharing git session by Ben !!

## 순서
#### 1. 원격 레포지토리 연결 
- 로컬에서 원격저장소로 push 할 파일들이 있는 위치로 이동하기 ( 프로젝트 최상위 폴더 )
- 원격 레포지토리 주소 복사
- git clone "원격 레포지토리 주소"
- git remote add origin "원격레포지토리 주소"
  
#### 2. 브랜치 생성
- git branch "브랜치 명"
  - 브랜치 생성 
- git branch -a 
  - 브랜치명 조회
- git branch -d "브랜치 명"
  -  브랜치 삭제 
- git checkout "브랜치 명" 
  - 브랜치 이동 

#### 3. 커밋하기
- git status 
  - 현재 로컬 저장소 상태확인
